# Zona Fit App 🏋️‍♀️

Aplicación fitness desarrollada con **Spring Boot**, **JPA** y **Lombok**.
Permite gestionar usuarios y rutinas mediante operaciones **CRUD** con persistencia en base de datos.

## Tecnologías
- Java 21 (JDK 21)
- Spring Boot
- JPA
- Lombok
- Maven
- MySQL


## Instalación
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/plazagustavo/Zona-Fit-App.git

2. Importar como proyecto Maven en tu IDE.
3. Configurar la base de datos en application.properties.
    ```bash
    spring.datasource.url=jdbc:mysql://localhost:3306/zonafit
    spring.datasource.username=root
    spring.datasource.password=tu_password

4. Ejecutar la aplicación desde la clase principal:
   ```bash
   src/main/java/gm/ZonaFitSpring/ZonaFitSpringApplication.java
